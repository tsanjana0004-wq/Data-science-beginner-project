**AI JOB MARKET ANALYSIS**
**Project Overview** : This project analyzes the trends of the job market using numpy,pandas and machine learning techniques.The goal is to identify the factors such as salary trends, job categories using real world data.
**Ojectives** : Analyzes job market data to find patterns, identify important skills for different roles, predict salary trends using machine learning.
**Technologies Used** : Python, Pandas, Numpy, Matplotlib and Scikit-learn
**Project Workflow** : Data Collection : Loaded dataset containing job-related information,Data Preprocessing : Handled missing values Cleaned and formatted data,Exploratory Data Analysis (EDA) : Visualized salary distribution,Analyzed job roles and skills
Machine Learning Models : Applied Random Forest for prediction,Used K-Means Clustering to group similar jobs
Insights & Results : Identified high-demand skills,Observed salary trends across roles.
**Results**: Certain technical skills show higher salary trends,Job roles can be grouped into clusters based on requirements,Machine learning models help in understanding job market patterns.

**Conclusion**: This project helped me understand the concepts like data analysis, machine learning and how AI can be applied to real world problems.
